Post Configuration

Recommended configuration steps to perform after installing Universal Controller and Universal Agent components.

Recommended Universal Controller Properties

Set these properties in the Universal Controller after installation (Services > Administration > Properties). Adjust values to match your environment.

Property Value
Administrator Email AddressRecipient(s) for Universal Controller system notifications. Expected: comma-separated list of email addresses. admin@email.com,admin2@email.com
Banner Background ColorHexadecimal color for the page banner in the UI. Expected: #RRGGBB (six hex characters) or "transparent". #4DC389
Bundle Exclude On Existence Picker DefaultDefault record types that will NOT be updated by a bundle promotion if they already exist on the target server. Expected: one or more record type selections. [Calendars, Custom Days, Variables, Credentials, Email Templates, Email Connections, Database Connections, SAP Connections, PeopleSoft Connections]
Change History EnabledControls whether users are prompted with a Change Description field when modifying records. Expected: true or false. true (recommended for prod)
Critical Path Calculations PermittedDetermines whether users can utilize the Critical Path feature. Expected: true or false. true
Exclude Holidays For Business DaysWhether holidays falling on business days are treated as non-business days. Expected: true or false. true
Resolvable Credentials PermittedEnables the use of resolvable credentials (allowing scripts/tasks to retrieve credential values at runtime). Expected: true or false. true
Retrieve Output Default Number Of LinesDefault number of lines retrieved when fetching task output; also acts as the limit for automatic retrieval. Expected: numeric value (default 100). 500
System IdentifierUser-selected name displayed in the User Task Bar to identify the environment (e.g., DEV01, PROD). Expected: free-form text string. Description of the environment (DEV01)
System Identifier Background ColorBackground color for the System Identifier field on the User Task Bar. Expected: color name (default: Black) or hex value. Black
System Identifier ColorText color for the System Identifier field on the User Task Bar. Expected: color name (default: White) or hex value. White
Universal Template Extension Maximum BytesMaximum allowed size (in bytes) for a Universal Template extension archive uploaded to the Controller. Expected: numeric byte value (e.g., 900000000 ≈ 900 MB). 900000000
Web Service Default Response ContentDefault response content type returned by Universal Controller Web Services when no format is specified by the client. Expected: JSON or XML. JSON
