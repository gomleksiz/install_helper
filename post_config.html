Post Configuration

Recommended configuration steps to perform after installing Universal Controller and Universal Agent components.

Recommended Universal Controller Properties

Set these properties in the Universal Controller after installation (Services > Administration > Properties). Adjust values to match your environment.

  1. Administrator Email Address = admin@email.com,admin2@email.comRecipient(s) for Universal Controller system notifications. Expected: comma-separated list of email addresses.
  2. Banner Background Color = #4DC389Hexadecimal color for the page banner in the UI. Expected: #RRGGBB (six hex characters) or "transparent".
  3. Bundle Exclude On Existence Picker Default = [Calendars, Custom Days, Variables, Credentials, Email Templates, Email Connections, Database Connections, SAP Connections, PeopleSoft Connections]Default record types that will NOT be updated by a bundle promotion if they already exist on the target server. Expected: one or more record type selections.
  4. Change History Enabled = true (recommended for prod)Controls whether users are prompted with a Change Description field when modifying records. Expected: true or false.
  5. Critical Path Calculations Permitted = trueDetermines whether users can utilize the Critical Path feature. Expected: true or false.
  6. Exclude Holidays For Business Days = trueWhether holidays falling on business days are treated as non-business days. Expected: true or false.
  7. Resolvable Credentials Permitted = trueEnables the use of resolvable credentials (allowing scripts/tasks to retrieve credential values at runtime). Expected: true or false.
  8. Retrieve Output Default Number Of Lines = 500Default number of lines retrieved when fetching task output; also acts as the limit for automatic retrieval. Expected: numeric value (default 100).
  9. System Identifier = Description of the environment (DEV01)User-selected name displayed in the User Task Bar to identify the environment (e.g., DEV01, PROD). Expected: free-form text string.
  10. System Identifier Background Color = BlackBackground color for the System Identifier field on the User Task Bar. Expected: color name (default: Black) or hex value.
  11. System Identifier Color = WhiteText color for the System Identifier field on the User Task Bar. Expected: color name (default: White) or hex value.
  12. Universal Template Extension Maximum Bytes = 900000000Maximum allowed size (in bytes) for a Universal Template extension archive uploaded to the Controller. Expected: numeric byte value (e.g., 900000000 ≈ 900 MB).
  13. Web Service Default Response Content = JSONDefault response content type returned by Universal Controller Web Services when no format is specified by the client. Expected: JSON or XML.

After applying the keystore and updating server.xml, restart Tomcat for the changes to take effect:
